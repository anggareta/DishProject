## Migrasi Database 
- Koneksi **SQL Server** harus tersedia dan sesuai dengan setting aplikasi pada file *appsettings.json*
- untuk migrasi database lakukan peritah ini, pada **Package Manager Console** di Visual Studio :
```powershell
PM> Update-Database
```
